## Build and Runtime Requirements
+ Xcode 9.0 or later
+ Swift 4 or later
+ iOS 11.0 or later
+ SpriteKit framework

## How to run 

1. Clone this repo
1. Open `ColorGame.xcodeproj`
1. Run the project on selected device or simulator
